Habilidades

Desarrollo frontend Jr.

He creado sitios web responsivos aplicando los conocimientos obtenidos durante el bootcamp de Tecnolochicas PRO.

SQL y mongo DB Jr.

He generado conjuntos de datos a partir de la manipulación de bases de datos relacionales y no relacionales para su posterior análisis, con el fin de ejecutar eficientes consultas SQL y NoSQL, aplicando los conocimientos obtenidos durante el curso de Ciencia de Datos Tecnolochicas PRO.

Programación Orientada a Objetos

He utilizado heramientas como Matlab para el desarrollo de una interfaz que permita realizar mediciones de impedancia de forma dinámica, en específico a baterias de litio mientras se encuentra cargando o descargando, en LabView desarrollé como estudiante una interfaz capaz de recibir y enviar información dependiendo ciertas respuestas de la simulación de un motor de CA en SolidWorks .

Investigación

En el proyecto de investigación se analizaron y discutieron formas de detección y clasificación de epilepsia a través de señales biológicas de EEG, utilizando una base de datos, y métodos de reconocimiento de patrones para poder determinar si padece o no el paciente epilepsia, con el fin de obtener de forma automática una clasificación de pacientes que padezcan epilepsia focal y de pacientes sanos.
